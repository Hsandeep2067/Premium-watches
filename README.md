# Premium Watch Website

A responsive, elegant website for a luxury watch brand built with HTML, CSS, and JavaScript.

## Features

- Fully responsive design that works on all devices
- Modern, elegant UI inspired by luxury watch brands
- Smooth scrolling navigation
- Interactive elements with hover effects
- Video modal for product showcase
- Animated elements that appear on scroll
- Mobile-friendly navigation menu
- Preloader for enhanced user experience
- Back-to-top button for easy navigation
- Decorative elements for visual enhancement
- Product badges for special offers

## Pages

1. **Home** - Hero section with call-to-action buttons
2. **Collection** - Showcase of featured watches with badges
3. **About** - Brand story and craftsmanship details
4. **Testimonials** - Customer reviews and feedback
5. **Contact** - Location information and contact details

## Technologies Used

- HTML5
- CSS3 (Flexbox and Grid layouts)
- JavaScript (ES6)
- Google Fonts
- Responsive design principles

## How to Run

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Alternatively, you can use a local server:
   - With Python: `python -m http.server 8000`
   - With Node.js: `npx serve`

## Customization

To customize the website:

1. **Change Colors**: Modify the color values in `styles.css`
2. **Update Content**: Edit the HTML content in `index.html`
3. **Modify Layout**: Adjust the CSS layout properties in `styles.css`
4. **Add Functionality**: Extend the JavaScript in `script.js`
